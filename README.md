
# 💜 Personal Bucket List Web App

A simple and aesthetic **Bucket List Web Application** built using HTML, PHP, and MySQL.  
Users can add, complete, and delete their goals easily.

---

## 🚀 Features

- ➕ Add new wishes with date  
- ✅ Mark wishes as completed  
- ❌ Delete wishes  
- 🎨 Modern purple gradient UI  
- 💎 Clean card-based layout  

---

## 🛠️ Tech Stack

- Frontend: HTML + CSS  
- Backend: PHP  
- Database: MySQL  
- Server: XAMPP  

---

## 📁 Project Structure


bucketlist/
│── index.html
│── action.php


---

## 🗄️ Database Setup

```sql
CREATE DATABASE bucketlist;

USE bucketlist;

CREATE TABLE goals (
    id INT AUTO_INCREMENT PRIMARY KEY,
    title VARCHAR(255),
    date DATE,
    status ENUM('pending','completed') DEFAULT 'pending'
);
